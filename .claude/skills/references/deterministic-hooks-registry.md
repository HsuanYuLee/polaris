> Source: EPIC-521 restraint mechanisms (2026-04-10), extended by DP-030 Phases 1вАУ3 (2026-04-24).

These mechanisms are enforced by **scripts + active hooks** (exit code driven), not behavioral rules. Blocking hooks physically block the action; advisory hooks inject deterministic guidance.
